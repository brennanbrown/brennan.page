## System Overview

The brennan.page homelab is a comprehensive self-hosting solution running on DigitalOcean with:

- **Infrastructure**: Docker, Caddy reverse proxy, automated backups
- **Management**: Portainer, FileBrowser, monitoring dashboard
- **Productivity**: Vikunja tasks, HedgeDoc notes, Linkding bookmarks
- **Community**: WriteFreely blog, Flarum forum, FreshRSS reader
- **Media**: Navidrome music streaming
